## 3. 推奨実行戦略

Codexレビューの結果、**以下の組み合わせが最も再現性が高い**:

```
主軸: Discord Bot as a Service（少数顧客・高単価・時間効率最良）
補助: ゲーミングマクロパッド（売れ筋1SKUのみ併用）
拡張: 電子工作キット（6ヶ月目以降の横展開候補）
```

### なぜこの組み合わせか

- **Discord Bot**: 3〜5社で月10万到達。40時間/月の制約との相性が最良
- **マクロパッド**: 趣味との親和性が高く、1SKU月3台でも補助収益として機能
- **電子工作キット**: マクロパッドと部品共通化が可能。横展開時の追加投資が小さい

## 13. Codexレビュー結果サマリー

### 第1回レビュー: 10候補の評価

- 再現性が高いのは「物販（キット/デバイス）」か「受託（Discord Bot）」
- コンシューマー向けアプリ単体で月10万は副業では非現実的
- 法的にグレーな領域（カスタムFW）は非推奨
- 推奨順: 電子工作キット > マクロパッド > Discord Bot > SEOツール > ワインアプリ

### 第2回レビュー: 上位3案の詳細評価

- **Discord Botが時間制約との相性最良**（少数顧客・高単価）
- マクロパッドは製造工数が時間上限にぶつかるリスク
- 電子工作キットは単価が低く必要販売数が多い → 拡張候補に格下げ
- 最終推奨順: Discord Bot > マクロパッド > 電子工作キット

### 致命的指摘の解消状況

- [x] カスタムFWの法的リスク → 候補から却下
- [x] Bot事業の集客リスク → 無料診断+実績公開の導線設計
- [x] Bot事業の24/7期待 → SLAで明文化
- [x] マクロパッドの個別カスタム化 → 3択制限 + 1SKU集中
- [x] 法務（特商法/PL法/契約書） → チェックリスト化
- [x] 税務（開業届/確定申告） → 初動アクションに組込
