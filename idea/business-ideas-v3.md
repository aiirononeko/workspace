# ビジネスアイデア提案書 v3

> 作成日: 2026-02-27
> レビュー済み: 投資家 / マーケター / PM / プロダクトデザイナー / レッドチーム / ペルソナユーザー（3名）

## 前提条件

- 個人開発者1名（副業）
- フルスタック: Next.js / TypeScript / Supabase
- プログラミングスクール講師経験あり
- Claude Code / AI活用に精通
- ブログ運営経験あり（microCMS）
- SNSフォロワーは少ない
- 目標: 月10万円以上の安定収入
- 制約: 時間の切り売りNG。プロダクト/アセット型で、スケーラブルな構造であること

## 評価プロセス

1. 初期案12個を策定
2. 6人の専門家エージェントが並行でフィードバック
   - **投資家**: 収益性・スケーラビリティ・参入障壁・リスク
   - **マーケター**: 市場規模・集客難易度・差別化・GTM戦略
   - **PM**: MVP工数・PMF難易度・ユーザーペイン・優先度
   - **プロダクトデザイナー**: UX複雑さ・デザイン差別化・個人実現可能性
   - **レッドチーム**: 失敗理由・競合見落とし・甘い仮定・致命的弱点・潰れる確率
   - **ペルソナユーザー**: 3名（28歳Web開発2年目 / 32歳テックリード / 25歳未経験転職志望）
3. フィードバック統合・ブラッシュアップ（12案→統合・選別）
4. Codexによるクロスレビュー
5. 最終版作成

**一言**: 日本でSaaSを立ち上げるために必要な全てが入ったオールインワンキット

| 項目 | 内容 |
|------|------|
| 概要 | Next.js + Supabase + Stripe JP ベースのSaaSボイラープレート。日本の法制度・決済事情への対応を支援するテンプレート群を同梱 |
| 価格 | 買い切り 19,800円 + 年間アップデート権 9,800円/年（初年度無料） |
| 月10万円達成 | 新規購入月5件（約10万円）+ アップデート権更新による継続収益で安定化 |
| MVP期間 | 4-6週（週末開発） |
| 免責 | 同梱テンプレートは法的助言ではない旨を明記。弁護士監修版は有料アドオンとして別SKU化を検討 |

**含まれる機能**:
- 認証（メール / Google / GitHub / LINE）
- 決済（Stripe JP対応、サブスク / 買い切り / 従量課金）
- 日本語メールテンプレート（Resend / SendGrid）
- 管理ダッシュボード（shadcn/ui）
- LP（日本語タイポグラフィ最適化、Noto Sans JP + Geist Mono）
- 特定商取引法に基づく表記テンプレート（※法的助言ではない旨の免責付き）
- プライバシーポリシーテンプレート（※同上）
- インボイス制度対応テンプレート（※同上）
- 日本語ドキュメント（Starlight / Nextra）

**6専門家の評価統合**:

| 視点 | 評価 | 根拠 |
|------|------|------|
| 投資家 | A | PMFが海外で証明済み（ShipFast月$50K+）。日本語版は空白地帯。限界費用ゼロ |
| マーケター | A | 「Next.js SaaS テンプレート 日本語」のSEO未開拓。ビルドインパブリックでX集客可能 |

| 指摘 | 対策 |
|------|------|

## Tier A: 並行して育てる（集客・ブランド基盤）

### 【3】技術ブログ + 有料コンテンツ

**位置づけ**: 全ビジネスの集客エンジン・信頼構築基盤。単体収益は副次的。

| 項目 | 内容 |
|------|------|
| 概要 | Claude Code / AI開発ワークフロー特化の技術ブログ。SEO + Zenn/note有料記事 + アフィリエイト |
| 収益貢献 | 単体で月3-5万円（SEO安定後）+ Launcher等への導線で間接+5万円以上 |
| 開発工数 | ゼロ（既存microCMS基盤 or Zenn/note利用） |

**全専門家が「必ずやるべき」と推奨した理由**:
- 他の全アイデアのマーケティング基盤として不可欠
- Claude Code活用記事は**今が先行者利益を取れるタイミング**
- 既にmicroCMSでブログ基盤がある
- 開発コストゼロで始められる

**レッドチーム指摘と対策**:

| 指摘 | 対策 |
|------|------|
| SEOは死にかけ（AI Overview等） | SEOに依存せず、X/SNS拡散を主軸に。実体験ベースで差別化 |
| AI生成コンテンツの氾濫 | 「Launcherを自分で作る過程」のリアルタイムドキュメンタリーが活路 |
| フォロワーが少ない | ブログ記事の要約をXで毎日投稿。3ヶ月でフォロワー500人が目標 |
| 有料記事は知名度がないと売れない | 無料記事で信頼構築→反響の大きいテーマを有料本に昇華 |

**コンテンツ戦略**:
1. 週2本のブログ記事（Claude Code活用、Next.js実践、SaaS開発の裏側）
2. 反響の大きい記事をZenn有料本（1,500-3,000円）にまとめ直す
3. プログラミングスクールアフィリエイトを自然に組み込む（講師経験を活かした信頼性の高い推薦）
4. **Launcher開発過程をそのまま記事化**（マーケティングコスト = 開発記録コスト）

## リスクマトリクス

| リスク | 影響度 | 発生確率 | 対策 |
|--------|-------|---------|------|
| ShipFast本家が日本語対応 | 高 | 中 | 日本固有機能（法制度対応テンプレート）で差別化。先行者利益で顧客基盤確保 |
| AI生成でボイラープレート不要に | 高 | 中 | プロダクション品質の設計パターン・ベストプラクティスで差別化 |
| Next.jsメジャーバージョンアップ | 中 | 高 | App Router安定版に追従。アップデートはコンテンツのネタに転用。年間アップデート権で保守費用を回収 |
| 集客が伸びない | 中 | 中 | 受託つなぎで資金確保しつつ、コンテンツ戦略を見直し |
| 燃え尽き（副業の限界） | 高 | 中 | 週18時間上限を厳守。受託は月1件まで |
| 価格競争（無料テンプレートの増加） | 中 | 高 | 「テンプレート」ではなく「日本市場ビジネスキット」として価値訴求 |
| セキュリティ（脆弱性混入・個人情報漏えい） | 高 | 中 | 依存パッケージの定期更新、Snyk/Dependabot導入、セキュリティチェックリスト同梱 |
| 法務（テンプレート利用による購入者トラブル） | 高 | 低 | 「法的助言ではない」免責を購入時・ドキュメント・LP全てに明記。弁護士監修版は別SKU |
| 税務（インボイス制度・確定申告） | 中 | 中 | freee/MFで自動管理。売上規模が拡大したら税理士に委託 |
| チャネル依存（X/YouTube/Udemy規約変更） | 中 | 中 | 自前メーリングリスト（Resend）を構築し、チャネルリスクを分散 |
| 保守・サポート負債の増大 | 中 | 高 | アップデート権の期限化（12ヶ月）、FAQ自動化、Discordコミュニティで購入者同士の助け合い促進。サポート窓口: GitHub Discussionsのみ、返信SLA: 72時間以内、個別実装相談は対象外。Phase 2以降は毎週3hを保守バジェットとして固定計上 |

## 撤退基準

- ローンチ後3ヶ月で累計販売10件未満 → ピボットまたは撤退

- サポート工数が週10時間を超える → 価格改定またはサポート体制見直し
